MOTEL Platform

Open Technology Data for Energy Models

MOTEL is a structured, ontology-ready technology database and workflow for transparent energy system modeling.

Repository Data Folder

What Is Included

  • Technology and process data in CSV tables
  • Controlled vocabularies and mappings
  • RDF export output for graph workflows
  • Notebook-based ingestion and harmonization steps

Key Folders

  • motel-db/: core tabular dataset
  • schema/: schema definitions
  • 3a_graphdb/: CSV to TTL conversion
  • 1_ingest/ and 2_harmonise/: data workflow notebooks
